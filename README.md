# Tomato Plant Disease Classification System

A machine learning and computer vision project designed to identify tomato leaf diseases using Deep Learning techniques with Convolutional Neural Networks (CNN). The model was developed with TensorFlow and Keras and trained on a real-world tomato leaf image dataset for accurate disease recognition.

---

## Introduction

This project aims to automate the detection of tomato plant diseases from leaf images. By applying image classification methods, the system can recognize different disease categories and healthy leaves, supporting smarter agricultural monitoring and crop protection.

---

## Main Features

- Deep learning based disease recognition
- CNN model for image classification
- Tomato leaf health analysis
- Dataset preprocessing and augmentation
- Training and testing using TensorFlow and Keras
- Implemented and executed in Google Colab
